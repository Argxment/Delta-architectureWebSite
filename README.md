Landing-page для архитектурного бюро, выполненный с акцентом на минимализм, функциональность и современную эстетику.

## 🛠 Особенности реализации
* **Адаптивный дизайн:** Сайт корректно отображается на всех устройствах (десктоп, планшеты, смартфоны).
* **Чистый код:** Использование семантических тегов HTML5 для лучшего SEO.
* **Интерактивность:** Плавная навигация и анимированные элементы интерфейса.
* **Структура:** Модульная организация CSS-файлов для удобства поддержки.

## 📁 Структура проекта
* `index.html` — Главная страница.
* `/shared/styles/` — Библиотека стилей и навигация.
* `/shared/js/` — Скрипты для интерактивных элементов.
* `/shared/assets/` — Графические материалы и изображения объектов.
